== Summary

Description:: List start and end milestones.
[NOTE]
.Syntax notes
====
In *USFM* versions < 3.2 it was not possible to explicitely identify the start and end of a xref:para:lists/index.adoc[list] (it needed to be inferred), or to assign attributes to a list. This milestone pair resolves this need. *USX* 3.2 also introduces a corresponding `<list>`...`</list>` container.
====
xref:ROOT:syntax-docs.adoc#_syntax[Syntax image:ROOT:help.svg[]]::
* *USFM:* ``++\list-s\*++``#__list content__##``++\list-e\*++``
* *USX:* ``++<list>++``#__list content__#``++</list>++``
Style Type:: xref:ms:index.adoc[Milestone]
Valid In:: `[xref:doc:index.adoc#doc-book-chapter-content[ChapterContent]]`
